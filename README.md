# agentic-library

A dump/library for my agentic AI work — reusable agents, schedule setup prompts, and supporting scripts.

## Layout

- `schedule-prompts/` — copy-paste prompts that tell Claude how to build and configure a recurring scheduled task. Drop a new `.md` file here for each schedule you want to be able to spin up.

## How to use a schedule prompt

1. Open the relevant file in `schedule-prompts/`.
2. Copy the prompt block into Claude.
3. Answer the configuration questions Claude asks.
4. Claude builds the scripts and registers the scheduled task.
